## Upgrading
- Unless stated here, any version may be upgraded to any newer version, but downgrades are not supported.

### To 0.0.3-DEV
- Passwords will not be saved, and therefore players will have to run the command again.
    - This was introduced to add user safety, as passwords will be hashed.
